🎵 Musik Studio Lite V4.5

Musik Studio Lite adalah aplikasi Digital Audio Workstation (DAW) berbasis HTML, CSS, dan JavaScript yang berjalan langsung di browser tanpa instalasi tambahan.

---

✨ Fitur Utama

- 🥁 Drum Pad
- 🎹 Piano Pad
- 🎸 Electric Guitar
- 🎸 Bass
- 🎤 Rekam Vokal
- ✨ FX Panel
- 🎼 MIDI Editor
- 🎼 Editor Musik (Timeline)
- 📥 Import Audio
- 📤 Export WAV / MP3
- 📁 Project Manager

🎯 Target

Musik Studio Lite dikembangkan sebagai DAW ringan yang dapat berjalan secara offline di browser maupun GitHub Pages, dengan fokus pada proses pembuatan musik menggunakan instrumen virtual, timeline editor, import audio, dan export WAV/MP3.
